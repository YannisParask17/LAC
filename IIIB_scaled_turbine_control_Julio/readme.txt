========================================================================================
Folder description
========================================================================================

control/	---->	.dll files for the controller

data/ 		---->	.dat files for structural and aerodynamic characteristics (inputs to hawc)

hawc2s_files/	---->	.htc files for different closed loop frequency and damping ratios in hawc2s (steady state sims)

htc/		---->	.htc files for stepped wind simulations (time-marching sims)

htc_master/ 	---->	.htc files for turbulent wind simulations (time-marching sims)

log/		---->	.log files from htc/ folder stepped wind simulations (time-marching sims)

res/		---->	.hdf5 result files from stepped wind simulations (time-marching sims)
